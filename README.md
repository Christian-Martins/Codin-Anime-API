<h1 align="center"> Codin'Anime API </h1>
<p align="center">Are you looking for an API to show you anime images with references to IT development? 
  Then <a href="https://codin-anime-api.herokuapp.com/picture">Codin'Anime API</a> is <strong>the solution</strong>! 
Codin'Anime is a simple API developed with ExpressJS that gives you a random image.</p>
<p align="center">=> Alpha 0.1.0</p>

<h1 align="center">Documentation</h1>

All pictures: `curl https://codin-anime-api.herokuapp.com/picture`

Random picture: `curl https://codin-anime-api.herokuapp.com/picture/random`

Specific category: `curl https://codin-anime-api.herokuapp.com/picture/category/:category`

=> *for exemple:* `curl https://codin-anime-api.herokuapp.com/picture/category/javascript`

Specific ID: `curl https://codin-anime-api.herokuapp.com/picture/id/:id`
